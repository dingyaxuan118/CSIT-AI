# SmartTutor - 智能作业辅导Agent

## 项目概述

本项目是CSIT5900课程的家庭作业辅导Agent，采用多阶段处理管道架构，重点关注可靠性和护栏机制。

### 核心功能

- **数学辅导**：微积分、代数、线性代数等大学一年级数学问题
- **历史辅导**：世界历史、重大历史事件、著名人物等
- **金融辅导**：投资学基础、金融市场、风险管理等
- **经济学辅导**：微观经济学、宏观经济学入门等
- **化学辅导**：有机化学、无机化学、元素周期表等
- **智能护栏**：自动识别并拒绝离题、冷门知识、危险内容
- **对话总结**：支持总结之前对话内容
- **多轮对话**：支持上下文连贯的多轮交流

### 技术架构

```
用户输入 → 护栏层(Guardrails) → 路由层(Router) → 学科Agent → 响应输出
  ↓           ↓                    ↓
 验证       意图分类             数学/历史/金融/经济/化学/拒绝
```

## 安装和运行

### 环境要求

- Python 3.8+
- Streamlit

### 安装步骤

1. 安装依赖：

```bash
pip install streamlit openai
```

2. 运行应用：

```bash
cd code
streamlit run app.py
```

3. 在浏览器中打开显示的URL（通常是 http://localhost:8501）

## 使用说明

### 基本使用

1. 在左侧边栏选择API来源（推荐使用mock模式进行测试）
2. 设置您的年级水平
3. 在输入框中输入数学或历史问题
4. 按Enter发送问题

### 支持的问题类型

#### 接受的问题示例

**数学问题**：
- "如何求 x+1=2 的解？"
- "√1000 是有理数吗？"
- "求 f(x)=x²·sin(x) 的导数"

**历史问题**：
- "法国大革命的经济原因是什么？"
- "法国的第一任总统是谁？"

**功能请求**：
- "总结我们之前的对话"

#### 拒绝的问题示例

**离题内容**：
- "帮我规划去东京的旅行"
- "推荐一部好看的电影"

**冷门知识**：
- "香港科技大学图书馆自动售货机的历史"

**危险内容**：
- "在街上放鞭炮会怎样？"

## 扩展说明

### 添加更多学科

可以在`app.py`中添加新的Agent来处理其他学科：

1. 在`code/`目录创建新的Agent文件（如`chemistry_agent.py`）
2. 在`app.py`中导入并创建实例
3. 在护栏系统的分类中添加新学科的判断

### 使用真实API

要使用真实的LLM API：

1. 设置环境变量：
   - OpenAI: `export OPENAI_API_KEY=your_key`
   - Azure: `export AZURE_OPENAI_API_KEY=your_key`

2. 在侧边栏选择相应的API来源
